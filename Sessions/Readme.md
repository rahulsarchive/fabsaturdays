The session covered the basiscs of 2D design using Rhinocerous. The partcipants learned to create 2D CAD models, desigining different types of wood joints, Learning about press-fit assemblies, Converting CAD to CAM using VcarvePro, Shopbot saftey, Basic operation of shopbot using controlpanel, generating G-code from CAD files, Cutting out a test piece to find press-fit tolerances.
